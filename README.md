## 🤖 Built with Agent-Based Development

This site showcases a working implementation of agentic AI in software development. Rather than traditional solo development, this project uses a coordinated system of AI agents:

- **Orchestrator** - Routes tasks and manages team coordination
- **Designer** - Creates UI/UX specifications and design systems  
- **Developer** - Implements features from design specifications
- **Tester** - Validates quality and accessibility compliance

**Why this matters:** This approach demonstrates practical AI workflow automation in real software delivery, moving beyond proof-of-concept to production implementation.

[**Learn more about the agent system →**](https://billyjhart.github.io/agents)

## 🛠 Technical Implementation

### Technology Stack
- **Framework:** Jekyll (GitHub Pages compatible)
- **Styling:** SCSS with responsive design system
- **Deployment:** GitHub Pages (automated)
- **Performance:** Optimized for resource-constrained environments
- **Accessibility:** WCAG AA compliance

### Architecture Highlights
- **Component-based design system** - Consistent, maintainable UI patterns
- **Performance-first approach** - Optimized for Raspberry Pi deployment
- **Semantic HTML structure** - Accessible and SEO-friendly
- **Mobile-responsive design** - Works across all device types

### Project Structure
```
├── _config.yml           # Jekyll configuration
├── _layouts/             # Page templates

├── _sass/                # SCSS styling system
├── assets/               # Images and compiled assets
├── index.md              # Homepage content

```
